# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Libex, please report it privately rather than opening a public issue.

**Email:** [swagar003@gmail.com](mailto:swagar003@gmail.com)

You can also use GitHub's [private vulnerability reporting](https://github.com/LibexHQ/Libex/security/advisories/new) to submit a report directly through the repository.

We'll acknowledge receipt as quickly as possible and work with you on a fix before any public disclosure.
